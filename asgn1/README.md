# Assignment 1

This program will have some vampires play a garlic game where they roll some dice. Every round, 1 vampire loses a life if they roll the lowest number. The game ends when only one vampire is left standing.

## Build
    easy way: $ make

    other way: $ clang -Wall -Wextra -Werror -Wpendantic -o vampire vampire.c

## Running

    $ ./vampire

## Cleaning
    easy way: $ make clean

    other way: $ rm -f vampire vampire.o
